# Services

Les services du bundle encapsulent la logique métier pour interagir avec l'API Stripe. Ils sont utilisés en interne par le trait `BillableTrait` mais peuvent également être injectés dans vos propres services.

## SubscriptionBuilder

Le `SubscriptionBuilder` permet de construire et créer des abonnements avec une interface fluide.

### Emplacement

```php
namespace CashierBundle\Service;

class SubscriptionBuilder
```

### Méthodes

#### Configuration des prix

```php
// Ajoute un prix à l'abonnement
$builder->price(string $price, int $quantity = 1, array $metadata = []): self

// Ajoute un prix metered (sans quantité fixe)
$builder->meteredPrice(string $price): self

// Définit la quantité du dernier prix ajouté
$builder->quantity(int $quantity): self
```

#### Configuration de l'essai

```php
// Définit une période d'essai en jours
$builder->trialDays(int $days): self

// Définit une date de fin d'essai
$builder->trialUntil(\DateTimeInterface $date): self

// Ignore l'essai (démarre immédiatement)
$builder->skipTrial(): self
```

#### Configuration des options

```php
// Ajoute des métadonnées
$builder->withMetadata(array $options): self

// Ajoute des options Stripe supplémentaires
$builder->withOptions(array $options): self

// Définit un coupon
$builder->withCoupon(?string $couponId): self

// Définit un code promotionnel
$builder->withPromotionCode(?string $code): self

// Définit le comportement de paiement
$builder->withPaymentBehavior(string $behavior): self
```

#### Configuration de la facturation

```php
// Ancré le cycle de facturation sur une date spécifique
$builder->anchorBillingCycleOn(\DateTimeInterface $date): self

// Définit des seuils de facturation
$builder->withBillingThresholds(array $thresholds): self

// Désactive la proratisation
$builder->noProrate(): self

// Active la proratisation (par défaut)
$builder->prorate(): self
```

#### Création

```php
// Crée l'abonnement
$builder->create(PaymentMethod|string|null $paymentMethod = null): Subscription
```

### Exemple d'utilisation

```php
$user->newSubscription('premium', 'price_premium_monthly')
    ->trialDays(14)
    ->withCoupon('SUMMER2024')
    ->quantity(3)
    ->create('pm_abc123');

// Abonnement avec plusieurs prix
$user->newSubscription('default')
    ->price('price_base', 1)
    ->price('price_addon', 2)
    ->create();

// Abonnement metered
$user->newSubscription('usage')
    ->meteredPrice('price_per_api_call')
    ->create();
```

## SubscriptionService

Le `SubscriptionService` gère les opérations sur les abonnements.

### Emplacement

```php
namespace CashierBundle\Service;

class SubscriptionService
```

### Méthodes

```php
// Crée un nouvel abonnement
$service->create(
    BillableInterface $billable,
    string $type,
    array $items,
    array $options = []
): Subscription

// Met à jour un abonnement
$service->update(Subscription $subscription, array $options = []): Subscription

// Annule un abonnement
$service->cancel(Subscription $subscription, bool $immediately = false): Subscription

// Reprend un abonnement annulé (en grace period)
$service->resume(Subscription $subscription): Subscription

// Change le prix d'un abonnement
$service->swap(Subscription $subscription, string $price, array $options = []): Subscription

// Met à jour la quantité d'un abonnement
$service->updateQuantity(Subscription $subscription, int $quantity): Subscription
```

## PaymentService

Le `PaymentService` gère les paiements ponctuels.

### Emplacement

```php
namespace CashierBundle\Service;

class PaymentService
```

### Méthodes

```php
// Effectue un paiement avec un moyen de paiement spécifique
$service->charge(
    BillableInterface $billable,
    int $amount,
    string $paymentMethod,
    array $options = []
): Payment

// Effectue un paiement avec le moyen par défaut
$service->pay(
    BillableInterface $billable,
    int $amount,
    array $options = []
): Payment

// Rembourse un paiement
$service->refund(string $paymentIntent, array $options = []): Refund

// Rembourse partiellement un paiement
$service->refundPartial(
    string $paymentIntent,
    int $amount,
    array $options = []
): Refund
```

## PaymentMethodService

Le `PaymentMethodService` gère les moyens de paiement.

### Emplacement

```php
namespace CashierBundle\Service;

class PaymentMethodService
```

### Méthodes

```php
// Ajoute un moyen de paiement au client
$service->add(BillableInterface $billable, string $paymentMethod): PaymentMethod

// Met à jour le moyen de paiement par défaut
$service->updateDefault(
    BillableInterface $billable,
    string $paymentMethod
): PaymentMethod

// Liste les moyens de paiement
$service->list(BillableInterface $billable, ?string $type = null): Collection

// Récupère le moyen de paiement par défaut
$service->default(BillableInterface $billable): ?PaymentMethod

// Vérifie si un moyen de paiement par défaut existe
$service->hasDefault(BillableInterface $billable): bool

// Supprime un moyen de paiement
$service->remove(BillableInterface $billable, string $paymentMethod): void
```

## InvoiceService

Le `InvoiceService` gère les factures.

### Emplacement

```php
namespace CashierBundle\Service;

class InvoiceService
```

### Méthodes

```php
// Liste les factures d'un client
$service->list(BillableInterface $billable, bool $includePending = false): Collection

// Crée une nouvelle facture
$service->create(BillableInterface $billable, array $options = []): Invoice

// Récupère une facture par son ID
$service->find(string $id): ?Invoice

// Récupère la prochaine facture à venir
$service->upcoming(BillableInterface $billable): ?Invoice

// Crée un item de facture (sans créer la facture)
$service->createInvoiceItem(
    BillableInterface $billable,
    array $invoiceItems
): void

// Ajoute un item à la prochaine facture
$service->tab(
    BillableInterface $billable,
    string $description,
    int $amount,
    array $options = []
): void

// Crée et paie une facture
$service->pay(Invoice $invoice): Payment

// Crée une facture pour un montant spécifique
$service->invoiceFor(
    BillableInterface $billable,
    string $description,
    int $amount,
    array $options = []
): Invoice

// Récupère les lignes d'une facture
$service->getInvoiceLines(string $invoiceId): array

// Supprime un item de facture
$service->deleteInvoiceItem(string $invoiceItemId): void
```

## CheckoutService

Le `CheckoutService` gère les sessions Checkout Stripe.

### Emplacement

```php
namespace CashierBundle\Service;

class CheckoutService
```

### Méthodes

```php
// Crée une session de paiement
$service->create(
    BillableInterface $billable,
    array $items,
    array $options = []
): Checkout

// Crée une session pour un paiement unique
$service->charge(
    BillableInterface $billable,
    int $amount,
    string $name,
    int $quantity = 1,
    array $options = []
): Checkout

// Crée une session d'abonnement
$service->createSubscription(
    BillableInterface $billable,
    array $items,
    array $options = []
): Checkout

// Génère une URL vers le portail de facturation
$service->billingPortal(
    BillableInterface $billable,
    ?string $returnUrl = null
): string

// Récupère une session de checkout
$service->findSession(string $sessionId): ?Checkout
```

## CustomerService

Le `CustomerService` gère les clients Stripe.

### Emplacement

```php
namespace CashierBundle\Service;

class CustomerService
```

### Méthodes

```php
// Crée un nouveau client Stripe
$service->create(BillableInterface $billable, array $options = []): string

// Met à jour un client Stripe
$service->update(BillableInterface $billable, array $options = []): void

// Récupère un client local par son ID Stripe
$service->find(string $stripeId): ?StripeCustomer

// Crée ou récupère l'ID Stripe d'un client
$service->createOrGetStripeId(
    BillableInterface $billable,
    array $options = []
): string

// Synchronise un client Stripe avec la base locale
$service->sync(
    Stripe\Customer $stripeCustomer,
    ?BillableInterface $billable = null
): void

// Génère le payload pour la création/mise à jour
$service->getStripePayload(BillableInterface $billable): array
```

## Injection de dépendances

Tous ces services peuvent être injectés dans vos propres services Symfony:

```php
use CashierBundle\Service\SubscriptionService;

class MyCustomService
{
    public function __construct(
        private readonly SubscriptionService $subscriptionService
    ) {}

    public function upgradeUser(User $user): void
    {
        $subscription = $user->subscription('default');
        if ($subscription) {
            $this->subscriptionService->swap(
                $subscription,
                'price_premium'
            );
        }
    }
}
```
