# Événements

Le bundle déclenche des événements Symfony lors des actions importantes liées à Stripe. Vous pouvez écouter ces événements pour réagir aux changements.

## Événements d'abonnement

### SubscriptionCreatedEvent

Déclenché lorsqu'un abonnement est créé.

#### Emplacement

```php
namespace CashierBundle\Event;

class SubscriptionCreatedEvent extends Event
```

#### Propriétés

```php
// L'abonnement créé
$event->subscription: Subscription
```

#### Méthodes

```php
// Récupère l'abonnement
$event->getSubscription(): Subscription
```

#### Exemple d'écouteur

```php
use CashierBundle\Event\SubscriptionCreatedEvent;

class SubscriptionCreatedListener
{
    public function __invoke(SubscriptionCreatedEvent $event): void
    {
        $subscription = $event->getSubscription();
        $user = $subscription->getCustomer()->getBillable();

        // Envoyer un email de bienvenue
        // Mettre à jour des statistiques
        // etc.
    }
}
```

#### Configuration

```yaml
# config/services.yaml
services:
    App\Listener\SubscriptionCreatedListener:
        tags:
            - { name: kernel.event_listener, event: CashierBundle\Event\SubscriptionCreatedEvent }
```

### SubscriptionDeletedEvent

Déclenché lorsqu'un abonnement est supprimé ou annulé.

#### Emplacement

```php
namespace CashierBundle\Event;

class SubscriptionDeletedEvent extends Event
```

#### Propriétés

```php
// L'abonnement supprimé
$event->subscription: Subscription
```

#### Méthodes

```php
// Récupère l'abonnement
$event->getSubscription(): Subscription
```

#### Exemple d'utilisation

```php
use CashierBundle\Event\SubscriptionDeletedEvent;

class SubscriptionDeletedListener
{
    public function __invoke(SubscriptionDeletedEvent $event): void
    {
        $subscription = $event->getSubscription();
        $user = $subscription->getCustomer()->getBillable();

        if ($subscription->onGracePeriod()) {
            // L'abonnement est encore actif jusqu'à la fin de la période
            $this->sendCancellationEmail($user);
        } else {
            // L'abonnement est immédiatement terminé
            $this->revokeAccess($user);
        }
    }
}
```

## Événements de paiement

### PaymentSucceededEvent

Déclenché lorsqu'un paiement réussi.

#### Emplacement

```php
namespace CashierBundle\Event;

class PaymentSucceededEvent extends Event
```

#### Propriétés

```php
// ID du client Stripe
$event->customerId: string

// ID du PaymentIntent
$event->paymentIntentId: ?string

// Montant en centimes
$event->amount: int

// Devise
$event->currency: string
```

#### Méthodes

```php
// Récupère l'ID du client
$event->getCustomerId(): string

// Récupère l'ID du PaymentIntent
$event->getPaymentIntentId(): ?string

// Récupère le montant en centimes
$event->getAmount(): int

// Récupère le montant en décimal
$event->getAmountInDecimal(): float

// Récupère la devise
$event->getCurrency(): string
```

#### Exemple d'utilisation

```php
use CashierBundle\Event\PaymentSucceededEvent;

class PaymentSucceededListener
{
    public function __invoke(PaymentSucceededEvent $event): void
    {
        $amount = $event->getAmountInDecimal();
        $currency = $event->getCurrency();

        $this->logger->info("Paiement de {$amount} {$currency} réussi");

        // Mettre à jour des statistiques
        // Débloquer du contenu
        // Envoyer une confirmation
    }
}
```

## Événements de webhook

### WebhookReceivedEvent

Déclenché lorsqu'un webhook Stripe est reçu, avant son traitement.

#### Emplacement

```php
namespace CashierBundle\Event;

class WebhookReceivedEvent extends Event
```

#### Propriétés

```php
// L'événement Stripe brut
$event->stripeEvent: \Stripe\Event
```

#### Méthodes

```php
// Récupère l'événement Stripe
$event->getStripeEvent(): \Stripe\Event
```

#### Utilisation

Cet événement est utile pour:
- Logger tous les webhooks entrants
- Filtrer certains événements
- Ajouter une logique de validation personnalisée
- Mettre à jour des statistiques en temps réel

```php
use CashierBundle\Event\WebhookReceivedEvent;
use Stripe\Event as StripeEvent;

class WebhookReceivedListener
{
    public function __invoke(WebhookReceivedEvent $event): void
    {
        $stripeEvent = $event->getStripeEvent();

        $this->logger->info('Webhook reçu', [
            'type' => $stripeEvent->type,
            'id' => $stripeEvent->id,
        ]);
    }
}
```

## Enregistrement des écouteurs

### Via la configuration (services.yaml)

```yaml
# config/services.yaml
services:
    App\Listener\SubscriptionCreatedListener:
        tags:
            - { name: kernel.event_listener, event: CashierBundle\Event\SubscriptionCreatedEvent }

    App\Listener\PaymentSucceededListener:
        tags:
            - { name: kernel.event_listener, event: CashierBundle\Event\PaymentSucceededEvent }

    App\Listener\WebhookReceivedListener:
        tags:
            - { name: kernel.event_listener, event: CashierBundle\Event\WebhookReceivedEvent }
```

### Via l'attribute PHP 8

```php
use Symfony\Component\EventDispatcher\Attribute\AsListener;
use CashierBundle\Event\SubscriptionCreatedEvent;

class SubscriptionCreatedListener
{
    #[AsListener]
    public function onSubscriptionCreated(SubscriptionCreatedEvent $event): void
    {
        // ...
    }
}
```

### Via une classe Subscriber

```php
use Symfony\Component\EventDispatcher\EventSubscriberInterface;
use CashierBundle\Event\SubscriptionCreatedEvent;
use CashierBundle\Event\SubscriptionUpdatedEvent;
use CashierBundle\Event\SubscriptionDeletedEvent;

class SubscriptionEventSubscriber implements EventSubscriberInterface
{
    public static function getSubscribedEvents(): array
    {
        return [
            SubscriptionCreatedEvent::class => 'onSubscriptionCreated',
            SubscriptionUpdatedEvent::class => 'onSubscriptionUpdated',
            SubscriptionDeletedEvent::class => 'onSubscriptionDeleted',
        ];
    }

    public function onSubscriptionCreated(SubscriptionCreatedEvent $event): void
    {
        // ...
    }

    public function onSubscriptionUpdated(SubscriptionUpdatedEvent $event): void
    {
        // ...
    }

    public function onSubscriptionDeleted(SubscriptionDeletedEvent $event): void
    {
        // ...
    }
}
```
