# Référence API

Cette documentation référence les services, modèles, entités et interfaces du bundle Stripe Cashier.

## Services

### CustomerService

Gestion des clients Stripe et synchronisation avec votre base de données locale.

**Responsabilités :**
- Création et mise à jour de clients Stripe
- Synchronisation des données clients
- Gestion du Customer Balance
- Import/Export de clients

**Méthodes principales :**
- `createCustomer()` - Créer un client Stripe
- `updateCustomer()` - Mettre à jour un client
- `syncCustomerDetails()` - Synchroniser les détails
- `importCustomer()` - Importer depuis un ID Stripe
- `exportCustomer()` - Exporter vers un ID Stripe

### PaymentService

Gestion des paiements, des refunds et des intents de paiement.

**Responsabilités :**
- Création de PaymentIntents
- Gestion des paiements one-time
- Gestion des refunds
- Vérification des paiements
- Webhook de paiement

**Méthodes principales :**
- `createPayment()` - Créer un paiement
- `refundPayment()` - Rembourser un paiement
- `confirmPayment()` - Confirmer un paiement
- `handlePaymentSucceeded()` - Gérer paiement réussi
- `handlePaymentFailed()` - Gérer paiement échoué

### SubscriptionService

Gestion complète des abonnements Stripe.

**Responsabilités :**
- Création et mise à jour d'abonnements
- Gestion des cycles de facturation
- Gestion des renouvellements
- Gestion des suspensions et annulations
- Migration d'abonnements

**Méthodes principales :**
- `createSubscription()` - Créer un abonnement
- `updateSubscription()` - Mettre à jour un abonnement
- `cancelSubscription()` - Annuler un abonnement
- `reactivateSubscription()` - Réactiver un abonnement
- `swapAndCancel()` - Échanger et annuler

### CheckoutService

Gestion de Stripe Checkout et du Billing Portal.

**Responsabilités :**
- Création de sessions Checkout
- Gestion du Checkout Experience
- Intégration du Billing Portal
- Gestion des sessions expirées
- Gestion des abandons de panier

**Méthodes principales :**
- `createCheckoutSession()` - Créer une session Checkout
- `createBillingPortalSession()` - Créer une session Billing Portal
- `expireCheckoutSession()` - Expirer une session
- `handleCheckoutCompleted()` - Gérer session complétée

### InvoiceService

Gestion des factures Stripe.

**Responsabilités :**
- Génération de factures
- Création d'objets d'écriture
- Gestion des factures échues
- Calcul des totaux
- Gestion des remises

**Méthodes principales :**
- `createInvoice()` - Créer une facture
- `createInvoiceItem()` - Créer un élément de facture
- `finalizeInvoice()` - Finaliser une facture
- `retryInvoice()` - Retenter le paiement d'une facture
- `listUpcomingInvoices()` - Lister les prochaines factures

### PaymentMethodService

Gestion des méthodes de paiement.

**Responsabilités :**
- Attachement de méthodes de paiement
- Gestion des SetupIntents
- Validation des méthodes de paiement
- Gestion des méthodes par défaut
- Migration entre méthodes

**Méthodes principales :**
- `attachPaymentMethod()` - Attacher une méthode de paiement
- `detachPaymentMethod()` - Détacher une méthode de paiement
- `setAsDefault()` - Définir comme méthode par défaut
- `retrievePaymentMethod()` - Récupérer une méthode de paiement
- `listPaymentMethods()` - Lister les méthodes de paiement

### TaxService

Gestion des taxes et calculs fiscaux.

**Responsabilités :**
- Calcul des taxes automatiques
- Gestion des TaxRates
- Configuration des paramètres fiscaux
- Synchronisation avec TaxJar/Avalara
- Gestion des taxes locales

**Méthodes principales :**
- `calculateTax()` - Calculer les taxes
- `createTaxRate()` - Créer un taux de taxe
- `listTaxRates()` - Lister les taux de taxe
- `updateTaxSettings()` - Mettre à jour les paramètres fiscaux
- `syncTaxJarSettings()` - Synchroniser avec TaxJar

## Interfaces

### BillableInterface

Interface pour les entités facturables.

```php
interface BillableInterface
{
    public function getCashierEmail(): string;
    public function getCashierName(): string;
    public function getCashierCurrency(): string;
}
```

### BillableEntityInterface

Interface pour les entités Doctrine facturables.

```php
interface BillableEntityInterface extends BillableInterface
{
    public function getStripeCustomer(): ?StripeCustomer;
    public function setStripeCustomer(?StripeCustomer $customer): void;
}
```

### InvoiceRendererInterface

Interface pour les renderers de facture personnalisés.

```php
interface InvoiceRendererInterface
{
    public function render(Invoice $invoice): string;
    public function supports(string $format): bool;
}
```

### WebhookHandlerInterface

Interface pour les handlers de webhook personnalisés.

```php
interface WebhookHandlerInterface
{
    public function handles(string $event): bool;
    public function handle(array $payload, WebhookReceivedEvent $event): void;
}
```
