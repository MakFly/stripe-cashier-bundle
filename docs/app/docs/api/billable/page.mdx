# Interface Billable

L'interface `BillableInterface` définit les méthodes que doivent implémenter les entités facturables (utilisateurs, organisations, etc.) pour interagir avec Stripe.

## Emplacement

```php
namespace CashierBundle\Contract;

interface BillableInterface
```

## Utilisation

Pour rendre une entité facturable, implémentez l'interface et utilisez le trait `BillableTrait`:

```php
use CashierBundle\Contract\BillableInterface;
use CashierBundle\Concerns\BillableTrait;

class User implements BillableInterface
{
    use BillableTrait;

    // ... votre code
}
```

## Méthodes de l'interface

### Gestion du client

#### `stripeId(): ?string`

Retourne l'identifiant Stripe du client.

```php
$stripeId = $user->stripeId(); // ex: "cus_abc123"
```

#### `hasStripeId(): bool`

Vérifie si le client a un identifiant Stripe.

```php
if ($user->hasStripeId()) {
    // Le client existe dans Stripe
}
```

#### `createAsStripeCustomer(array $options = []): string`

Crée un nouveau client Stripe et retourne son identifiant.

```php
$stripeId = $user->createAsStripeCustomer([
    'email' => 'user@example.com',
    'name' => 'John Doe',
]);
```

#### `updateStripeCustomer(array $options = []): void`

Met à jour les informations du client dans Stripe.

```php
$user->updateStripeCustomer([
    'email' => 'newemail@example.com',
    'name' => 'John Smith',
]);
```

#### `createOrGetStripeCustomer(array $options = []): string`

Crée un client Stripe s'il n'existe pas, sinon retourne l'identifiant existant.

```php
$stripeId = $user->createOrGetStripeCustomer();
```

#### `asStripeCustomer(): ?StripeCustomer`

Retourne l'objet Stripe Customer complet.

```php
$customer = $user->asStripeCustomer();
```

### Gestion des abonnements

#### `subscriptions(): Collection`

Retourne tous les abonnements du client.

```php
$subscriptions = $user->subscriptions();
```

#### `subscription(string $type = 'default'): ?Subscription`

Retourne l'abonnement d'un type spécifique.

```php
$subscription = $user->subscription('premium');
```

#### `subscribed(string $type = 'default', ?string $price = null): bool`

Vérifie si le client est abonné à un plan spécifique.

```php
if ($user->subscribed('default', 'price_monthly')) {
    // L'utilisateur a un abonnement actif
}
```

#### `onTrial(string $type = 'default', ?string $price = null): bool`

Vérifie si le client est en période d'essai.

```php
if ($user->onTrial()) {
    // L'utilisateur est en période d'essai
}
```

#### `onGenericTrial(): bool`

Vérifie si le client bénéficie d'un essai générique (pas lié à un abonnement spécifique).

```php
if ($user->onGenericTrial()) {
    // Essai générique actif
}
```

#### `newSubscription(string $type, string|array $prices = []): SubscriptionBuilder`

Crée un nouveau builder pour un abonnement.

```php
$subscription = $user->newSubscription('default', 'price_monthly')
    ->trialDays(14)
    ->create();
```

### Gestion des moyens de paiement

#### `hasDefaultPaymentMethod(): bool`

Vérifie si le client a un moyen de paiement par défaut.

```php
if ($user->hasDefaultPaymentMethod()) {
    // Un moyen de paiement est configuré
}
```

#### `defaultPaymentMethod(): ?PaymentMethod`

Retourne le moyen de paiement par défaut.

```php
$paymentMethod = $user->defaultPaymentMethod();
```

#### `addPaymentMethod(string $paymentMethod): PaymentMethod`

Ajoute un nouveau moyen de paiement au client.

```php
$paymentMethod = $user->addPaymentMethod('pm_abc123');
```

#### `updateDefaultPaymentMethod(string $paymentMethod): PaymentMethod`

Définit le moyen de paiement par défaut.

```php
$paymentMethod = $user->updateDefaultPaymentMethod('pm_xyz789');
```

#### `paymentMethods(?string $type = null): Collection`

Retourne tous les moyens de paiement du client, filtrés par type si spécifié.

```php
$cards = $user->paymentMethods('card');
```

### Paiements

#### `charge(int $amount, string $paymentMethod, array $options = []): Payment`

Effectue un paiement immédiat.

```php
$payment = $user->charge(1000, 'pm_abc123', [
    'description' => 'Achat ponctuel',
]);
```

#### `pay(int $amount, array $options = []): Payment`

Effectue un paiement avec le moyen de paiement par défaut.

```php
$payment = $user->pay(1000, [
    'description' => 'Achat ponctuel',
]);
```

#### `refund(string $paymentIntent, array $options = []): Refund`

Rembourse un paiement.

```php
$refund = $user->refund($paymentIntent->id());
```

### Factures

#### `invoices(bool $includePending = false): Collection`

Retourne toutes les factures du client.

```php
$invoices = $user->invoices(); // Factures payées uniquement
$allInvoices = $user->invoices(true); // Inclut les factures en attente
```

#### `invoice(array $options = []): Invoice`

Crée une nouvelle facture pour le client.

```php
$invoice = $user->invoice();
```

#### `upcomingInvoice(): ?Invoice`

Retourne la prochaine facture à venir.

```php
$upcoming = $user->upcomingInvoice();
```

#### `tab(string $description, int $amount, array $options = []): self`

Ajoute un élément à la facture du client (sans créer la facture immédiatement).

```php
$user->tab('Service additionnel', 500);
```

#### `invoiceFor(string $description, int $amount): Invoice`

Crée une facture pour un montant spécifique.

```php
$invoice = $user->invoiceFor('Service additionnel', 500);
```

### Checkout

#### `checkout(array $items): Checkout`

Crée une session Checkout Stripe.

```php
$checkout = $user->checkout([
    ['price' => 'price_monthly', 'quantity' => 1],
]);
```

#### `checkoutCharge(int $amount, string $name, int $quantity = 1): Checkout`

Crée une session Checkout pour un paiement unique.

```php
$checkout = $user->checkoutCharge(1000, 'Produit spécial');
```

### Solde client

#### `balance(): string`

Retourne le solde du client.

```php
$balance = $user->balance(); // ex: "-500" (négatif = crédit)
```

#### `creditBalance(int $amount, ?string $description = null): CustomerBalanceTransaction`

Ajoute un crédit au solde du client.

```php
$transaction = $user->creditBalance(1000, 'Remboursement');
```

#### `debitBalance(int $amount, ?string $description = null): CustomerBalanceTransaction`

Débite le solde du client.

```php
$transaction = $user->debitBalance(500, 'Frais supplémentaires');
```

### Portail de facturation

#### `billingPortalUrl(?string $returnUrl = null): string`

Génère une URL vers le portail de facturation Stripe.

```php
$url = $user->billingPortalUrl('https://example.com/return');
```

### Taxes

#### `taxRates(): array`

Retourne les taux de taxe applicables au client.

```php
$taxRates = $user->taxRates();
```

#### `isTaxExempt(): bool`

Vérifie si le client est exempté de taxes.

```php
if ($user->isTaxExempt()) {
    // Pas de taxes à appliquer
}
```
