# Entités

Le bundle utilise des entités Doctrine pour stocker les informations synchronisées avec Stripe. Ces entités sont mappées aux objets Stripe correspondants.

## StripeCustomer

L'entité `StripeCustomer` représente un client Stripe dans votre base de données locale.

### Emplacement

```php
namespace CashierBundle\Entity;

class StripeCustomer
```

### Propriétés

| Propriété | Type | Description |
|-----------|------|-------------|
| `id` | `int\|null` | Identifiant primaire |
| `billable` | `BillableEntityInterface` | Entité facturable associée (User, etc.) |
| `stripeId` | `string` | Identifiant Stripe du client (unique) |
| `pmType` | `string\|null` | Type de moyen de paiement par défaut |
| `pmLastFour` | `string\|null` | 4 derniers chiffres du moyen de paiement |
| `trialEndsAt` | `DateTimeImmutable\|null` | Date de fin de l'essai générique |
| `createdAt` | `DateTimeImmutable` | Date de création |
| `updatedAt` | `DateTimeImmutable` | Date de dernière mise à jour |

### Méthodes

#### Getters

```php
$customer->getId(): ?int
$customer->getBillable(): BillableEntityInterface
$customer->getStripeId(): string
$customer->getPmType(): ?string
$customer->getPmLastFour(): ?string
$customer->getTrialEndsAt(): ?DateTimeImmutable
$customer->getCreatedAt(): DateTimeImmutable
$customer->getUpdatedAt(): DateTimeImmutable
```

#### Setters

```php
$customer->setBillable(BillableEntityInterface $billable): self
$customer->setStripeId(string $stripeId): self
$customer->setPmType(?string $pmType): self
$customer->setPmLastFour(?string $pmLastFour): self
$customer->setTrialEndsAt(?DateTimeImmutable $trialEndsAt): self
```

#### Méthodes d'état

```php
// Vérifie si le client est en période d'essai
$customer->onTrial(): bool

// Vérifie si le client est en essai générique
$customer->onGenericTrial(): bool
```

#### Relations

```php
// Récupère tous les abonnements du client
$customer->getSubscriptions(): Collection

// Ajoute un abonnement
$customer->addSubscription(Subscription $subscription): self

// Supprime un abonnement
$customer->removeSubscription(Subscription $subscription): self
```

## Subscription

L'entité `Subscription` représente un abonnement Stripe synchronisé localement.

### Emplacement

```php
namespace CashierBundle\Entity;

class Subscription
```

### Constantes de statut

```php
Subscription::STATUS_ACTIVE = 'active'
Subscription::STATUS_CANCELED = 'canceled'
Subscription::STATUS_INCOMPLETE = 'incomplete'
Subscription::STATUS_INCOMPLETE_EXPIRED = 'incomplete_expired'
Subscription::STATUS_PAST_DUE = 'past_due'
Subscription::STATUS_PAUSED = 'paused'
Subscription::STATUS_TRIALING = 'trialing'
Subscription::STATUS_UNPAID = 'unpaid'
```

### Propriétés

| Propriété | Type | Description |
|-----------|------|-------------|
| `id` | `int\|null` | Identifiant primaire |
| `customer` | `StripeCustomer` | Client associé |
| `type` | `string` | Type d'abonnement (ex: "default", "premium") |
| `stripeId` | `string` | Identifiant Stripe de l'abonnement (unique) |
| `stripeStatus` | `string` | Statut Stripe de l'abonnement |
| `stripePrice` | `string\|null` | Identifiant du prix principal |
| `quantity` | `int\|null` | Quantité de l'abonnement |
| `trialEndsAt` | `DateTimeImmutable\|null` | Date de fin de l'essai |
| `endsAt` | `DateTimeImmutable\|null` | Date de fin (si annulé) |
| `createdAt` | `DateTimeImmutable` | Date de création |
| `updatedAt` | `DateTimeImmutable` | Date de dernière mise à jour |

### Méthodes d'état

```php
// Vérifie si l'abonnement est valide (actif, essai ou grace period)
$subscription->valid(): bool

// Vérifie si l'abonnement est actif
$subscription->active(): bool

// Vérifie si l'abonnement est en période d'essai
$subscription->onTrial(): bool

// Vérifie si l'abonnement est en grace period (annulé mais encore actif)
$subscription->onGracePeriod(): bool

// Vérifie si l'abonnement a été annulé
$subscription->canceled(): bool

// Vérifie si l'abonnement est terminé
$subscription->ended(): bool

// Vérifie si l'abonnement est incomplet
$subscription->incomplete(): bool

// Vérifie si l'abonnement est en retard de paiement
$subscription->pastDue(): bool

// Vérifie si l'abonnement est incomplet et expiré
$subscription->incompleteAndExpired(): bool

// Vérifie si l'abonnement n'est pas en grace period
$subscription->notOnGracePeriod(): bool

// Vérifie si l'abonnement n'est pas en essai
$subscription->notOnTrial(): bool

// Vérifie si l'abonnement est récurrent (pas d'essai, pas de grace period)
$subscription->recurring(): bool

// Vérifie si l'abonnement est en pause
$subscription->paused(): bool

// Vérifie si l'abonnement est en grace period de pause
$subscription->onPausedGracePeriod(): bool

// Vérifie si l'abonnement n'est pas en pause
$subscription->notPaused(): bool

// Vérifie si l'abonnement n'est pas en pause ou en grace period de pause
$subscription->notPausedOrOnPausedGracePeriod(): bool
```

### Méthodes de relation

```php
// Récupère tous les items de l'abonnement
$subscription->getItems(): Collection

// Ajoute un item à l'abonnement
$subscription->addItem(SubscriptionItem $item): self

// Supprime un item de l'abonnement
$subscription->removeItem(SubscriptionItem $item): self
```

## SubscriptionItem

L'entité `SubscriptionItem` représente un item d'abonnement (pour les abonnements avec plusieurs prix).

### Emplacement

```php
namespace CashierBundle\Entity;

class SubscriptionItem
```

### Propriétés

| Propriété | Type | Description |
|-----------|------|-------------|
| `id` | `int\|null` | Identifiant primaire |
| `subscription` | `Subscription` | Abonnement parent |
| `stripeId` | `string` | Identifiant Stripe de l'item (unique) |
| `stripeProduct` | `string` | Identifiant du produit Stripe |
| `stripePrice` | `string` | Identifiant du prix Stripe |
| `quantity` | `int\|null` | Quantité (null pour les prix metered) |
| `meterId` | `string\|null` | Identifiant du mètre pour les prix metered |
| `meterEventName` | `string\|null` | Nom de l'événement pour les prix metered |
| `createdAt` | `DateTimeImmutable` | Date de création |
| `updatedAt` | `DateTimeImmutable` | Date de dernière mise à jour |

### Méthodes

#### Getters

```php
$item->getId(): ?int
$item->getSubscription(): Subscription
$item->getStripeId(): string
$item->getStripeProduct(): string
$item->getStripePrice(): string
$item->getQuantity(): ?int
$item->getMeterId(): ?string
$item->getMeterEventName(): ?string
$item->getCreatedAt(): DateTimeImmutable
$item->getUpdatedAt(): DateTimeImmutable
```

#### Setters

```php
$item->setSubscription(Subscription $subscription): self
$item->setStripeId(string $stripeId): self
$item->setStripeProduct(string $stripeProduct): self
$item->setStripePrice(string $stripePrice): self
$item->setQuantity(?int $quantity): self
$item->setMeterId(?string $meterId): self
$item->setMeterEventName(?string $meterEventName): self
```

### Exemple d'utilisation

```php
// Récupérer un abonnement
$subscription = $user->subscription('default');

// Vérifier son état
if ($subscription->active()) {
    echo "Abonnement actif";
}

if ($subscription->onTrial()) {
    $daysLeft = $subscription->getTrialEndsAt()->diff(new \DateTime())->days;
    echo "Encore {$daysLeft} jours d'essai";
}

// Parcourir les items
foreach ($subscription->getItems() as $item) {
    echo "Prix: {$item->getStripePrice()}, Quantité: {$item->getQuantity()}";
}
```
