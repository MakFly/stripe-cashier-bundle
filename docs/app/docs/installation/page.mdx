# Installation

## Via Composer

```bash
composer require cashier/cashier-bundle
```

## Enregistrer le bundle

Ajoutez le bundle dans `config/bundles.php` :

```php
return [
    // ...
    CashierBundle\CashierBundle::class => ['all' => true],
];
```

## Variables d'environnement

Ajoutez ces variables à votre `.env` :

```env
STRIPE_KEY=pk_test_xxx
STRIPE_SECRET=sk_test_xxx
STRIPE_WEBHOOK_SECRET=whsec_xxx
```

## Configuration

Créez `config/packages/cashier.yaml` :

```yaml
cashier:
    key: '%env(STRIPE_KEY)%'
    secret: '%env(STRIPE_SECRET)%'

    webhook:
        secret: '%env(STRIPE_WEBHOOK_SECRET)%'
        tolerance: 300  # secondes (défaut)
        events:
            - customer.subscription.created
            - customer.subscription.updated
            - customer.subscription.deleted
            - invoice.paid
            - invoice.payment_failed

    currency: usd               # défaut : usd
    currency_locale: en         # défaut : en

    invoices:
        renderer: CashierBundle\Service\InvoiceRenderer\DompdfInvoiceRenderer
        options:
            paper: letter
            remote_enabled: false
```

## Route webhook

Importez la route dans `config/routes.yaml` :

```yaml
cashier_webhooks:
    resource: '@CashierBundle/Resources/config/routes/webhook.yaml'
```

Cela expose le endpoint `POST /cashier/webhook`.

## Configurer votre entité

Votre entité utilisateur doit implémenter `BillableEntityInterface` et utiliser `BillableTrait` :

```php
use CashierBundle\Contract\BillableEntityInterface;
use CashierBundle\Concerns\BillableTrait;
use Doctrine\ORM\Mapping as ORM;

#[ORM\Entity]
class User implements BillableEntityInterface
{
    use BillableTrait;

    #[ORM\Id]
    #[ORM\GeneratedValue]
    #[ORM\Column]
    private ?int $id = null;

    #[ORM\Column(length: 255)]
    private string $email;

    #[ORM\Column(length: 255, nullable: true)]
    private ?string $name = null;

    public function getId(): ?int { return $this->id; }
    public function getEmail(): string { return $this->email; }
    public function getName(): ?string { return $this->name; }
}
```

## Base de données

Créez les tables nécessaires :

```bash
php bin/console doctrine:migrations:diff
php bin/console doctrine:migrations:migrate
```

Deux tables seront créées :
- `cashier_stripe_customers` — Clients Stripe liés à vos entités
- `cashier_subscriptions` — Abonnements actifs
- `cashier_subscription_items` — Items d'abonnement (plans multiples)
